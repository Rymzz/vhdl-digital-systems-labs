# VHDL Digital Systems Labs

This repository contains a collection of VHDL labs I completed as part of my digital systems coursework.

The goal of these labs was to practice designing, simulating, and testing digital circuits using VHDL. The labs cover combinational logic, sequential circuits, finite state machines, and testbench-based validation.

## Topics Covered

- Combinational logic design
- Sequential circuits
- Registers and counters
- Finite state machines
- Testbenches and simulation
- Digital system design using VHDL

## Repository Structure

| Folder | Description |
|--------|-------------|
| `lab00` | Introductory VHDL lab |
| `lab01` | Basic digital logic design |
| `lab02` | VHDL design and simulation practice |
| `lab03` | Sequential logic practice |
| `lab04` | Digital systems lab work |
| `lab05` | Final VHDL lab work |

## Technologies Used

- VHDL
- Digital logic design
- Simulation tools
- FPGA-oriented design workflow

## What I Learned

Through these labs, I practiced writing structured VHDL code, designing digital circuits, creating testbenches, and debugging hardware-oriented logic.

These labs helped me understand how digital systems are built and tested before being deployed on hardware.

## Academic Context

These labs were completed as academic work. Some starter files or lab instructions may come from course material, but the completed work reflects my learning and practice.
